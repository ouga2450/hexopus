# Hexopus（ヘキソパス）

> タスクを6つに絞って、ただ片づけるだけ。

誰でも続けられるをコンセプトにした、意思決定を極力減らした習慣アプリ。優先順位順に1タスクずつ表示し、「達成」「スキップ」「今日はここまで」だけで操作できる。

---

## 技術スタック

| | 技術 |
|---|---|
| フロントエンド | Angular |
| バックエンド | Rails 8 API |
| DB | PostgreSQL |
| 認証 | JWT |
| 起動 | Docker Compose |

## 開発環境のセットアップ

```bash
docker compose up
```

- フロントエンド: http://localhost:4200
- バックエンド: http://localhost:3000

## 主な機能

- タスクを最大6件まで登録・並び替え
- 優先順位の高いタスクを1件ずつ表示
- 「達成」「スキップ」「今日はここまで」の3択で操作
- 今日の達成数・スキップ数を振り返り
